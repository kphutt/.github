# Security Policy

## Reporting a Vulnerability

If you find a security vulnerability in any of my repositories, please report it responsibly:

1. **Do not** open a public GitHub issue
2. Use GitHub's [private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability) feature, or email the address in my GitHub profile
3. Include steps to reproduce and potential impact

I'll acknowledge receipt within 48 hours and aim to address confirmed vulnerabilities promptly.
